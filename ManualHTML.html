    1.INTRODUCCION A HTML
        1.1 Crear el documento html

    Para poner el tipo de documento 
        < !doctype html >

    Para poner cualquier etiqueta
        < ejemplo >

    Luego para cerrar cualquier etiqueta
        

    Si queremos poner cualquier atributo dentro de la etiqueta pero para cerrarlo no hay que poner el atributo
        < ejemplo atributo=valor>  ….. < /ejemplo>

    Todo documento HTML  tiene esta organización:
    
    < !doctype html >
    < html lang=es>		(El elemento HTML indica el comienzo de nuestro documento, hay que cerrarlo al final. 
                        También se puede indicar el idioma del documento dentro de esta etiqueta. )

        < head>			(El elemento head es la cabecera de nuestro documento)
    < /head>
        < body>			(El elemento body es el cuerpo de nuestro documento)
        < /body>
    < /html>
    
           1.2 DENTRO DE LA ETIQUETA < head>
    
    Dentro de esta se pone el titulo del documento y para ejecutarlo queda así:
        < head>
        < title> Titulo documento 
        < /head>

    También se ponen los metadatos para identificar el formato de codificación o el autor:
       < head>
       < title> Titulo documento 
       < meta charset=UTF-8>
       < meta name=autor content=nombre>
       
    
            1.3 DENTRO DE LA ETIQUETA < BODY>

            Elemento heading
        Este elemento pone el texto que se encuentra dentro como un encabezado. 
             Dependiendo de la importancia que queramos darle, lo pondremos en un nivel u otro, empezando por el primer nivel < h1> hasta el sexto nivel < h6>
                  
    < h1> Este texto aparecera mas grande que < /h1>
                  
    < h6> este text ya que este es de tipo 6< /h6> 
    

            Elemento article
        Este elemento se usa cuando se quiere poner una publicación de un foro o revista que tiene contenido independiente
            < article> publicación < /article>
           
            Elemento section
        Este elemento se usa cuando se escribe una agrupación temática. Las etiquetas de section van dentro de otra etiqueta article.
            < article>
                < section> … < /section>
            

            Elemento nav
        Este elemento representa una sección de una página cuyo propósito es proporcionar enlaces de navegación
        < nav>
            < p>Texto
            < /nav>
           
            Elemento aside
        Este elemento representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento
        < aside>
            < p>Texto< /p>
        < /aside>
                
            Elemento footer
        Este elemento crea un pie de página
        < footer>
         < address>
                  Texto que queramos poner
           < /address>
        < /footer>
           
            Elemento address
        Este elemento sirve para que el creador ponga información de contacto
            < adress>
                Tu direccion
                Tu correo
            < /adress>        
    
        
    
    2. AGRUPACIÓN

    2.1 Párrafo y texto preformateado 
    
    Para escribir un párrafo, usaremos la etiqueta < p>
        < p> Esto es un párrafo < /p>
    Para escribir un texto ya formateado como queremos, es decir, escribir en nuestro HTML el texto de la forma que queremos que se vea, usaremos la etiqueta < pre>. A parte, utilizará una letra monoespaciada.
        < pre> Este texto 
            aparecerá tal y como   
                lo estoy escribiendo < /pre> 
        
        2.2 Salto de línea  y líneas separadoras

    Para escribir un salto de línea, sin usar el elemento de preformateado, usaremos la etiqueta < br>. Esta etiqueta no necesita cerrarla y se puede poner donde se quiera.
        El texto comienza aquí

    

    

    

    
        Y termina aquí 

    Y si queremos escribir con saltos de línea en los párrafos, usaremos < wbr>

    Para escribir una línea separadora, usaremos la etiqueta < hr>. Al igual que la etiqueta de salto de línea, no hace falta el cierre de esta y se puede poner donde se quiera.
    
        Debajo hay una línea 
        
    
        Arriba hay una línea 

    
        
    	2.3 Listas
    Para crear elementos en una lista, utilizaremos la etiqueta < li>, podemos crear todos los que queramos.
    Si queremos hacer una lista ordenada usaremos la etiqueta < ol>, en cambio, si queremos hacerlas desordenadas usaremos < ul>.
    Por otro lado si la queremos definir,  podemos definir la descripción con la etiqueta < dl>, la etiqueta < dt> define los nombres y la etiqueta < dd> describe cada nombre.

      

       

      Nick:
      

      289807
      

      Comida:
      

      Pizza
      

      Color:
      

      Rojo
      

      
    También podemos añadir argumentos en las listas ordenadas, estos son:
        < ol reversed>
        < ol start”numero”>
        < ol type=”tipo”>

        
    


3. ELEMENTOS PARA EL TEXTO
	3.1 Tipos de letra (Resaltado)
    Hay siete tipos de resaltado, por un lado están: bastardilla, negrita y subrayado. Se ponen junto a la palabra o frase que se quiere resaltar. 
        Bastardilla: < i>
        Negrita: < b>
        Subrayada: < u>
    Cuando queramos dejar de resaltar las palabras, habrá que cerrar la etiqueta.

    Ejemplo: 
     Este texto tiene negrita, bastardilla y subrayado

    Otro elemento de resaltado es el énfasis < EM>, recalca una parte.
    Para decir que una parte es importante usaremos < strong>
    Si escribimos la “letra pequeña” de algo usaremos < small>
    Por último, si queremos poner que el contenido es irrelevante, usaremos < s>
     
     
    3.2 Hiperenlace
    Para añadir un enlace en nuestro documento, usaremos la etiqueta < a href=”La dirección que queramos>Texto que queramos resaltar que nos lleve al enlace< /a>.
    Tenemos la opción de poder movernos dentro de nuestro documento.
    Usaremos dos etiquetas, donde queramos que nos lleve escribiremos < elto  id=anclaje>nos llevará a este sitio< /a>
    Y por otro lado, para decir que desde esa palabra nos lleve usaremos < a href=#anclaje>cuando pinchemos nos llevará aqui< /a>
    También podemos hacer lo mismo para que nos lleve al inicio del documento poniendo:
        < a href=#top>

    4. IMÁGENES
    Podemos introducir imagenes, de todos los tipos, con la etiqueta < src>. A esta etiqueta podemos añadir argumentos como el tamaño de la imagen(width y height)  y un texto alternativo (alt).
        Ejemplo:
            < img src="ladirecciondelafoto.elformato" width=500 height=500 alt=explicacion de la foto>

    5.MAPAS DE IMAGEN

    Para crear un mapa de imagen, usaremos una referencia < img usemap=#referencia>
    y el elemento del nombre donde lo referenciamos < map name=referencia>, dentro de esta última, pondremos la etiqueta < area> con los atributos, shape, coords, href, alt, download, target.
    Quedando así:
        < img usemap=#referencia>
        < map name=referencia>
        < area (aquí dentro los valores que les pongamos junto a los atributos)>

    Si usamos el argumento coordenadas, tendremos que asignarle un tipo entre rect (las coordenadas van por las esquinas), circle(aquí añadiremos la x y la y del centro junto al radio)
    o poly(cuando queramos hacer una figura que no sea ninguna de las anteriores, usaremos xN y yN)

           6. TABLAS
    Podemos crear y añadir tablas, para ello tendremos que usar 8 tipos de etiquetas.
        < table> 		Para crear la tabla
        < tr>			Para crear una fila, seguido se pondrá el texto que queramos que tenga
        < td>			Para crear una columna, seguido se pondrá el texto que queramos que tenga
        < th>			Para poner un encabezado a la tabla
        < thead>		Sirve para poner un encabezado a la tabla
        < tbody>		Sirve para añadir cuerpo a la tabla
        < tfoot>		Sirve para crear el sumerio de la tabla
        < caption>		Sirve para incluir leyendas en la tabla

    Como a casi todos los elementos, le podremos añadir atributos, estos atributos afectan al borde, la expansión de la columna y la expansión de la fila.
        Para el borde usaremos border (en la etiqueta < table>)
        Para la expansión en columna usaremos rowspan (en las etiquetas < td> y < th>)
        Para la expansión en la fila usaremos colspan (en las etiquetas < td> y < th>)

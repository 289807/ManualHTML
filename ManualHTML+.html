    1.FORMULARIO
    Esta etiqueta se usa para hacer controles interactivos para que un usuario envíe cierta información

    Etiqueta: 
    	< form>
    Dentro de esta etiqueta podemos añadir los argumentos:

	    action(La URI de un programa que procesa la información que se ha pedido anteriormente)

	    method(el método con el que se enviará el formulario)
	    	get (se envia la URI) 
	    	post(los datos se incluyen en el cuerpo del formulario)

	    enctype(es el tipo de MIME del contenido para enviar el formulario)
           application/x-www-form-urlencoded
          multipart/form-data
         text/plain

	    accept-charset(lista de codificación de caracteres que el servidor acepta)
    


        2.ETIQUETA
    Representa una etiqueta para la interfaz que recibe el usuario.
    Etiqueta:
    	< label>
    Dentro de la etiqueta podemos añadir el argumento:
	    for (se pondrá para identificar el ID del elemento)


        3.ENTRADA
    Usaremos este control para meter controles interactivos
    Etiqueta:
	    < input>
    Dentro de esta se pueden poner varios atributos
        type(tipo de control que vamos a mostrar)  

    Dentro de la etiqueta type podremos:

    Para crear elementos de texto pondremos type=text
    
    Para enviar el formulario pondremos  type=submit
   
    Para controlar que se introducen números usaremos  type=number
	    También tendremos que ponerle dentro un valor mínimo (min) y máximo (max) y un número real (value).
    
    Para crear casillas de selección usaremos  type=checkbox
	    También tendremos que ponerle dentro unos valores como un nombre (name), visibilidad (value)  y si queremos que de primeras este verificada (checked)  
    
    Para crear botones de opción y se despliega una pestaña type=radio
	    También usaremos unos atributos como el nombre(name), la posición (value) y si queremos que de primeras este una por defecto (checked) 
   
    Para poner una casilla donde haya que poner una contraseña usaremos type=password
   
    Para poner un cuadro de texto para introducción textos de búsqueda type=search
    
    Para introducción un número telefónico type=tel
   
    Para introducir un campo que edite la URL type=url
    
    Para introducir un campo donde poner un correo electrónico type=email
    
    Si no necesitamos que se introduzcan números pero no importa el valor exacto usaremos type=range
   
    Si no es muy relevante que el control se vea en la pantalla usaremos type=hidden
    
    Podemos añadir imágenes type=image src=nombreimagen.formato
    
    Si queremos que el  usuario pueda seleccionar un archivo usaremos type=file
   
    Podemos añadir botones sin comportamiento type=button
   
    Y si el usuario se equivoca le podemos dar la opción de resetear el formulario con type=reset
   
    Podemos añadir el introducir una fecha con type=date
    	Dentro podemos poner valores mínimos (min) y máximo(max)
    
    Si solo queremos el mes y el año type=month
    
    Para introducir solo el numero de semana del año type=week
   
    Controlar que solo sea un valor de tiempo type=time
    
    Y por último en fechas, si queremos poner el tiempo actual type=datetime-local
    
    Si queremos cambiar el colore de algo type=color

            5.MENÚS
    Para crear menús, usaremos dos etiquetas principales < select> y < option>

    Dentro de la etiqueta < select> tendremos los atributos:
    	name (para aplicarle un ID)
    	size (representa el número de filas que serán visibles)
    	múltiple(si se pueden seleccionar varios elementos del menú)

    Dentro de la etiqueta < option> tendremos los atributos 
	    value(el valor que será enviado en el formulario)
	    selected (sí desde un principio está seleccionada)
	    disabled (si está establecido, el elemento no se puede seleccionar)
	    label (determina el significado de la opción)

    También tenemos otra etiqueta grande pero no es tan principal < optgroup> que sirve para dividir una opción en dos valores
       
        Ejemplo de codigo:
    < select name=select> 
    < option value=1> 1< /option>
    < optgroup label=dos>
    < option value=dosa selected>2a< /option>
    < option value=dosb>2b< /option>
    < /optgroup>
    < option value=3> 3< /option> 
        < /select>

        6.EDITOR DE TEXTO
    Un control para la edición multitarea de un texto
    Etiqueta:
    	< textarea>
    Dentro de este usaremos una serie de atributos
	    name (nombre que el asignaremos)
    	cols( anchura del control de texto
	    rows(número de líneas visibles en el control)
	    maxlength/minlength (máximo y mínimo número de caracteres)


            7.LISTA CON VALORES POSIBLES
    Cuando creamos una lista, podemos ponerle las etiquetas de < datalist> y < option>

    En < datalist> pondremos el atributo id para darle un nombre cuando lo referenciamos
    En < option> crearemos cada una de las opciones con el atributo < value>
        


        8.MARCOS
    Sirven para representar un contexto de navegación anidado que incrusta otra página en la nuestra.

    Etiqueta: 
    	< iframe>
    Dentro del etiqueta pondremos los atributos:
	    src (la dirección de la página)
	    srcdoc (el contenido de la página que contiene)
	    sandbox (aplica restricciones adicionales https://developer.mozilla.org/es/docs/Web/HTML/Elemento/iframe 
	    allowfullscreen (para definir si se pone pantalla completa o no)
        width/height(el tamaño que queramos que tenga la ventana)
        
    
        
    
    

    


            9.MULTIMEDIA
    Podemos añadir multimedia como videos o música con las etiquetas de < audio> y < video>

    Para < audio> usaremos el atributo src para poner la dirección junto a formato
 

    Para < video> usaremos los atributos
    	src (para poner la dirección)
    	height/width (para las dimensiones de la ventana que queramos que salga para la reproducción del video)
    	type (para ponerle el formato del video)
    	poster (para crear una miniatura en la ventana del video)

    Dentro de la etiqueta < video> ponemos < video controls> y después una serie de atributos que nos dejaran crear una especie de menu de reproduccion, pero podemos ponerle otros como  autoplay, muted, loop, preload como funciones para aplicarle al video 

    Otro recurso que podemos usar es el < track> que se usa para poner títulos, descripciones, capítulos o metadatos en los medios de audio y video.
	
    Dentro habrá atributos como:
	    kind(para decir el tipo de track)
	    	subtitle
		    captions
		    descriptions 
		    chapters
		    metadata
        label(para darle una etiqueta)
